# Rss Term

Get rss feeds to your terminal

# Install

    git clone https://github.com/Allan-1/rss-term.git

    cd rss-term

    npm install

# Usage

    $node . --help

    usage:
      node . -u [rssurl]

    example:
      node . -u https://media.rss.com/espionagevr/feed.xml
